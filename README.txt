Скрипт для инвентаризации однотипного оборудования с нуля.

в файл inv_numbers.txt помещаются инвентарные номера (один номер на строку)
Zebra QLn320 должна стоять как принтер по умолчанию, этикетка будет распечатана мгновенно.
PDF файлы с парами серийный номер-инвентарный номер создаются в каталоге qrcodes.
Дополнительно пара серийный-инвентарный номер дописывается в файл inv_data.txt
